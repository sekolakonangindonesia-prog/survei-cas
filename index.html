Sinkronisasi Data Terpusat

Agar data dari semua petugas lapangan bisa dikumpulkan di satu tempat, daftarkan dulu JSONBin.io (gratis, tidak perlu server). Setelah dapat API Key dan Bin ID, masukkan di bawah.

1
Buka jsonbin.io → Register gratis → Login
2
Klik API Keys di menu kiri → Salin Secret Key
3
Klik Bins → New Bin → isi dengan [] → Create → Salin BIN ID
4
Paste ke kolom di bawah → Simpan → Bagikan file HTML ini ke semua petugas

Cara Pakai

1
Simpan file HTML ini ke HP atau share link hosting
2
Petugas isi form, klik Simpan & Kirim Data
3
Supervisor buka tab Data, klik Sinkron (pojok kanan atas) untuk tarik data terbaru
4
Download CSV dari tab Ekspor untuk laporan
