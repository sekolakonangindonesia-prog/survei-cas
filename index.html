Sinkronisasi Data Terpusat

Agar data dari semua petugas lapangan bisa dikumpulkan di satu tempat, daftarkan dulu JSONBin.io (gratis, tidak perlu server).

1
Buka jsonbin.io → Register gratis → Login
2
Klik API Keys → Salin Secret Key
3
Klik Bins → New Bin → isi [] → Create → Salin BIN ID
4
Paste ke kolom di bawah → Simpan → Bagikan file HTML ini ke semua marketing

Cara Pakai

1
Simpan file HTML ini ke HP atau share link hosting
2
Marketing isi form, klik Simpan & Kirim Data
3
Koordinator buka tab Data, klik Sinkron untuk tarik data terbaru
4
Download CSV atau PDF dari tab Ekspor untuk laporan
